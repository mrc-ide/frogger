## Benchmarking

The benchmark script can be used to run the different iterations fo the model via R to get timings. Initially this script was used to compare leapfrog and frogger timings. Since we want to replace leapfrog with frogger, we'll no longer be able to run them separately. This document has some timings from two machines for posterity.
